Fat-Free

This is one of the most simplified, lightweight and one of the top PHP frameworks currently. It weighs less than 50KB and is considered as one of the lightweight PHP frameworks.

The main features of URL router, multilingual application support and cache engine have been used for the creation of this powerful framework.

The framework comprises of a variety of plugins, a powerful queue library, an amazing ORM, painless routing and a simple authentication. The framework has been built using Symfony components.

Kohana

Kohana was built with PHP OOP in mind and provides the many advantages like visibility protection, abstracts, automatic class loading, interfaces, overloading and singletons.

However, you can use this lightweight PHP framework only you have PHP 5 or updated versions.

FuelPHP

This is one of the top PHP frameworks which is written in PHP 5.3.  It uses a Cascading File Effect.

It’s mian advantages are- RESTful implementation, HMVC implementation, URL routing system, form and data validation, vulnerability protection and caching system.

Flight

This is a micro-framework and quite extensible as well. The documentation of this framework is simple and one can easily learn it.

The framework is not recommended for larger web application projects, as it is relatively simpler.

Zikula

This framework is based on Symfony and Doctrine. The framework is suitable for all kinds of big and small projects.

The Framework provides the advantage of being integrated with Font Awesome 4 and Bootstrap 3. The framework comes with a flexible theme system, ModuleStudio, chat and document creator galleries.

The platform is an open source one and is absolutely free.

 

PHPixie

Build back in 2012 with the aim of introducing a high-performance PHP framework for read-only websites, PHPixie is quite new. Just like FuelPHP it implements the HMVC architecture design pattern and was built using independent components. These components are available for use without the framework itself and are 100% unit tested requiring minimum dependencies.

Slim

In search of a PHP framework that enables you to quickly write simple yet powerful web applications and application programming interfaces (APIs)? Say hello to Slim. Slim is a PHP micro framework and is minimalistic in design. Thus, it’s excellent for small web applications where a full-stack PHP framework is not really needed.

Furthermore, it’s used by many PHP developers to develop RESTful APIs and web services. It’s packed with rich features like URL routing, client-side HTTP caching, session and cookie encryption, etc.

Li3

Lithium makes use of PHP namespaces to create a powerful plugin architecture. Almost every component of the PHP framework is replaceable. Lithium can return JSON and many other formats from the same actions that render templates increasing productivity in many cases. It handles a “type” key in the route which allows it to render various responses with different Content-Types. This is incredibly useful and easy to extend. This combined with closures in routes makes Lithium a very good framework for building a RESTful API.

Nette

Nette is open source PHP framework and is distributed under the MIT license. It is a set of PHP 7 standalone components that make up the framework, rated as the 3rd most popular in the world. It puts extra emphasis on productivity, clean code, and security.
